How to Navigate spotNgo

Get step-by-step paths with high-tech visual cues and A* intelligent routing.

01

Explore Campus

Drag the blueprint map to pan around. Scroll with mouse wheel or use controls to zoom dynamically on multi-floor zones.

02

Set Departure

Double-Click any node or room on the map to set your starting location. You can also select it via the search panel.

03

Set Destination

Right-Click any node to set your destination, or use the drop-down selector to choose your target classroom or lab.

04

Laser Flow Route

Click Find Path. Follow the glowing neon laser line and step-by-step directions to navigate buildings and floors.

Project

About the Project

spotNgo was created to solve indoor routing complexities across the Heritage Institute of Technology campus, utilizing advanced mathematical graphs and modern web visual systems.

Why spotNgo?

Large campus infrastructures with multiple buildings (Central Block, CME, ICT) and stacked floors can make navigating complex hallways, stairs, and laboratories difficult for new students, faculty, and visitors.

This application implements an advanced A* Pathfinding Algorithm with integrated distance weighting and floor-change penalties. Our responsive visual vector engine renders high-performance laser guidance overlays directly onto your screen.

About spotNgo

spotNgo is a state-of-the-art indoor navigation system designed specifically for the Heritage Institute of Technology campus.

Powered by advanced A* pathfinding and an interactive multi-floor vector engine, it provides seamless turn-by-turn routing between classrooms, labs, halls, and gates.

How to Use

  • Drag the map to pan around.
  • Scroll to zoom in and out.
  • Double-Click a room to set it as your Start point.
  • Right-Click a room to set it as your Destination.
  • Floor Tabs automatically change when viewing multi-floor paths.
